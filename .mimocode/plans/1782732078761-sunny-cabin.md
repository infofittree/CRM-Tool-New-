- API URL baked into build: `VITE_API_URL=https://crm-fittree.up.railway.app`

## Steps
1. Build frontend with production API URL
2. Create `.htaccess` for SPA routing on Hostinger
3. Upload web/dist/ contents to Hostinger public_html via File Manager or FTP

## Files
- `web/.htaccess` — SPA routing rules for Apache (Hostinger)
- `web/dist/` — Production build output

## SPA Routing on Hostinger
Apache needs rewrite rules so all routes serve `index.html`:
```
RewriteEngine On
RewriteBase /
RewriteRule ^index\.html$ - [L]
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule . /index.html [L]
```
