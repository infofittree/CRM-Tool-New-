# Complete Task Workflow — Permanent Root Cause Fix

## Root Cause Found

After tracing the entire chain, the **primary root cause** is a silent failure in `handleCompleteTask()` when the follow-up POST fails. The error is caught but never shown to the user — the button just does nothing.

**Secondary issue**: The wizard rendering condition `showActivityWizard && selectedTask && wizardFollowupId` can fail if any of the three state updates don't complete before the next render.

## The Complete Chain (verified working)

1. User clicks "Complete Task" → `handleCompleteTask()`
2. If `followup_id` exists → `setWizardFollowupId(id)` + `setShowActivityWizard(true)` → wizard opens
3. If `followup_id` is null → POST `/followups` → on success: set states → wizard opens
4. Wizard renders with `followupId=wizardFollowupId`
5. User completes steps → `handleSubmit()` → `completeActivity(followupId, payload)`
6. Backend: `complete_activity` → `activity_engine.process_activity_wizard()` → DB commit
7. Response → `onComplete()` → invalidates queries → UI refreshes

## Fixes Applied

### Fix 1: Add error feedback when follow-up creation fails
**File:** `web/src/pages/Tasks.tsx`
- In `handleCompleteTask`, if the POST fails, show an error state instead of silently failing
- Add `const [error, setError] = useState("")` state
- Show error message in the drawer when the wizard can't open

### Fix 2: Add loading state to Complete Task button
**File:** `web/src/pages/Tasks.tsx`
- Add `const [creatingFollowup, setCreatingFollowup] = useState(false)`
- Show spinner on the button while creating the follow-up
- Prevent double-clicks

### Fix 3: Add error feedback in ActivityWizard
**File:** `web/src/components/ActivityWizard.tsx`
- In `handleSubmit`, catch errors and show them to the user
- Currently errors are only logged to console

### Fix 4: Add success feedback after wizard completion
**File:** `web/src/pages/Tasks.tsx`
- In `onWizardComplete`, the success message is already set
- Ensure the drawer stays open long enough for the user to read it
